##wanandroid
基于鸿洋大神玩android提供的API打造的一款android客户端，做这个应用的出发点主要是：
>1、练习kotlin；    
>2、练习组件化；          
>3、练习使用Android Architecture Components搭建MVVM框架；         
>4、集合当前流行的一些框架：rxjava + retrofit + dagger2 + glide + rxcache + Arouter + leakcarary + BaseRecyclerViewAdapterHelper等；
